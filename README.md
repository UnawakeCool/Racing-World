So i need to tell you how to launch the game
Step 1, press Windows key + R and type "cmd" in the text box. Step 2, type "winget install Python.Python.3.11
Step 3, if it asks you if you agree to the terms, type "Y". Step 4, after it finishes setting up (the Python setup window closes). Then download the file. Step 5, press Windows key + R and type "cmd" in the text box. Step 6, type "cd Documents", press enter, and after that "python race_world_finalrelease3.py". Step 7, press enter. You're in! To play it after you downloaded the file, do the exact same thing except remove step 1,2, 3, and 4. You noticed, this is for Windows. Sadly, i do not know how to start it up on Mac.
